 Drone Fleet Tracking System

Real-time drone fleet tracking system built with ASP.NET Core, SignalR, SQLite and Leaflet.

 Features

- Real-time drone telemetry tracking
- Live map integration with Leaflet
- SignalR realtime communication
- SQLite database integration
- Dynamic drone marker updates
- Route path tracking
- Risk score calculation algorithm
- Live drone simulation
- Multi-drone support

---

 Technologies

 Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- SignalR

 Frontend
- HTML
- CSS
- JavaScript
- Leaflet.js
